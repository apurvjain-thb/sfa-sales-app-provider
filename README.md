# SFA Mobile App Prototype

A fully interactive Android mobile application prototype built for Pharmaceutical Field Sales Representatives.

The application serves as the execution layer on top of an intelligent CRM and beat planning engine that automatically generates daily doctor engagement plans based on doctor tiering, personas, historical interactions, digital engagement signals, and territory mapping.

## Key Features

### Dashboard

* Sales activity overview
* Visit performance metrics
* Pending task tracking
* Daily beat summary

### Today's Plan

* AI generated daily beat plan
* Agenda and calendar views
* Doctor intelligence and visit insights
* Guided visit execution workflow

### Contacts

* Doctor directory
* Search, filters, and sorting
* Historical interactions
* Relationship insights

### Tasks

* Follow up and action management
* Status tracking
* Priority management
* Task timelines and notes

### Profile

* Shift management
* Account settings
* Issue reporting

## Technology

* HTML5
* CSS3
* Vanilla JavaScript

Single file architecture with no external dependencies.

## Design Principles

* Mobile first
* Execution focused
* Enterprise healthcare UX
* Optimized for field sales productivity

## Purpose

This prototype demonstrates how pharmaceutical sales teams can execute smarter doctor engagement strategies through AI assisted beat planning, contextual doctor intelligence, and streamlined field workflows.
